# 🪐 Quantum Quest — Learn Quantum Computing from Scratch

A fun, interactive, click-everything course that takes you from **"what even is a qubit?"**
all the way to **writing real quantum programs and running them on a real quantum computer.**
Made to be understandable if you're about 12 years old (or 92 — curiosity is the only requirement!).

## ▶ How to start

- **Easiest:** double-click **`Start Quantum Quest.bat`** (Windows) — it opens the course in your browser.
- Or open **`index.html`** in any web browser (Chrome, Edge, Firefox...).

No installing, no internet needed for the main course. Your progress is saved automatically.

## 📚 The 11 lessons

0. Welcome  
1. Bits vs Qubits — the spinning coin  
2. Superposition — being two things at once  
3. Measurement — why looking changes everything  
4. Gates — the buttons that control qubits  
5. The Bloch Sphere — a qubit's secret globe  
6. Entanglement — spooky linked qubits 👻  
7. Circuits — snap gates together like LEGO  
8. Your first quantum trick — interference ✨  
9. Write real code — Python + Qiskit, run on real hardware  
10. Build your own systems — projects & next steps  

## 🧠 Every lesson has

- **Interactive toys** (spin qubits, build circuits, measure entangled pairs)
- **Quizzes** with instant feedback
- A **progress bar** that remembers where you are
