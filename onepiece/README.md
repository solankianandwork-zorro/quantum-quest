# 🏴‍☠️ Grand Line Quantum — the One Piece edition

A pirate-themed reskin of **Quantum Quest**. Same real quantum simulator and the same
interactive widgets, retold as a One Piece adventure across the Grand Line:

| Quantum idea | One Piece flavor |
|---|---|
| Qubit | a spinning **Devil Fruit Berry** |
| Superposition | **Devil Fruit awakening** (two things at once) |
| Measurement | **Observation Haki** (looking collapses it) |
| Gates | **combat moves** (X = Gomu Gomu Flip, H = Mixer, Z = Twister) |
| Bloch sphere | the **Log Pose globe** |
| Entanglement | **Nakama bonds** (linked crewmates) |
| Circuit | a **battle formation** |
| Interference | a **secret combo** (waves cancel/pile up) |

### Extra dynamic touches
- Animated ocean (rolling waves), drifting clouds, and a bobbing ship in the backdrop
- Log Pose progress meter
- A treasure-confetti shower (🪙💰👑💎) when you claim the Quantum One Piece

### ▶ How to start
- Open **`index.html`** in any browser, or play it online at
  **https://solankianandwork-zorro.github.io/quantum-quest/onepiece/**

No install needed. Your voyage progress is saved automatically (separate from the original course).
The Python/Qiskit track lives in the parent `../python/` folder.
